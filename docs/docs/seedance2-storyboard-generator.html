1. 项目介绍

Seedance 2.0 故事转视频工作流是一款专业的AI视频制作工具，旨在实现从小说/故事到多集视频的全流程自动化。该项目利用 Claude Code、Agent Skills、Seedance 2.0、Nana Banana Pro、Markdown、Python 等技术，为用户提供完整的AI视频制作解决方案。

为什么做这个？

因为 Seedance 2.0 已经改变了一切，需要新的剧本剧情制作工具，更好的利用新AI的能力和更方便的制作短剧。

核心特性

  • 🎯 三件套工具：Claude Code剧本创作 + Nana Banana Pro生图 + Seedance 2.0视频生成
  • 📝 四幕剧本结构：起承转合的专业剧本框架，AI自动生成完整故事
  • 🎨 素材编号系统：角色(C01-C99)、场景(S01-S99)、道具(P01-P99)的统一管理
  • ⏱️ Seedance格式：自动生成时间轴格式提示词，支持0-15秒分段描述
  • 🔗 视频延长功能：使用视频延长实现各集无缝衔接，创建连续视频系列
  • 🎭 风格一致性：统一风格前缀确保所有素材视觉一致性
  • 📚 示例项目：林冲水浒传、聂风风云、莫泊桑项链等实战案例
  • 🎬 完整工作流：构思主题 → 写剧本 → 生成素材描述 → 生图 → 写分镜脚本 → 逐集生成视频

技术栈

  • Claude Code：剧本创作与分镜脚本生成
  • Agent Skills：Claude Code技能系统，自动化工作流
  • Seedance 2.0：即梦AI视频生成平台
  • Nana Banana Pro：AI图像生成工具（可选其他生图AI）
  • Markdown：剧本和分镜脚本格式
  • Python：数据处理和自动化脚本

2. 安装配置

在开始使用 Seedance 2.0 故事转视频工作流 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows、macOS 或 Linux
  • 网络连接：稳定的互联网连接
  • Claude Code：已安装并配置 Claude Code
  • AI工具账户：Seedance 2.0、Nana Banana Pro（或类似生图工具）

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/Seedance2-Storyboard-Generator.git
cd Seedance2-Storyboard-Generator
  2. 配置 Claude Code Skill： 
    cp -r .claude/skills/seedance-storyboard-generator ~/.claude/skills/
  3. 准备AI工具账户：
    • Seedance 2.0：访问 即梦官网 注册账户
    • Nana Banana Pro：或选择其他AI生图工具（如Midjourney、Stable Diffusion等）
  4. 验证安装：在 Claude Code 中测试技能是否正常工作

3. 使用方法

基本使用流程

  1. 准备故事素材：选择要改编的小说、故事或创作原创内容
  2. 启动Claude Code：打开终端或编辑器，启动 Claude Code
  3. 调用技能：使用自然语言描述你的需求，例如： 
    请帮我为"林冲风雪山神庙"创作视频剧本
  4. 生成剧本：AI会自动生成四幕结构的剧本
  5. 生成素材清单：AI会创建编号的角色、场景、道具清单
  6. 生图：使用AI生图工具生成素材图片
  7. 生成分镜脚本：AI会生成Seedance 2.0格式的时间轴提示词
  8. 生成视频：在Seedance 2.0平台使用提示词生成视频

技能调用示例

# 基本剧本创作
请为"小红帽"创作5集视频剧本

# 指定风格
请为"三国演义赤壁之战"创作视频剧本，风格要求：中国古典水墨风格

# 指定集数和时长
请为"灰姑娘"创作10集视频，每集30秒

# 续写剧本
请为第5集生成分镜脚本，需要与第4集无缝衔接

4. 工作流程详解

4.1 剧本创作

将源材料改编为四幕结构剧本（起承转合）：

  • 第一幕（起）：介绍背景和人物，建立故事世界
  • 第二幕（承）：发展冲突，人物面临挑战
  • 第三幕（转）：情节转折，高潮冲突
  • 第四幕（合）：解决冲突，故事收尾

4.2 素材规划

创建编号的角色(C)、场景(S)、道具(P)生成提示词：

  • C01-C99：角色（多角度、多状态）
  • S01-S99：场景/地点
  • P01-P99：道具/物品

素材编号示例

C01 林冲·正面全身
C02 林冲·侧面部特写
C03 林冲·持枪姿势
S01 沧州草料场·雪景·全景
S02 山神庙·内景
P01 花枪·长枪
P02 酒葫芦

4.3 图像生成

使用 Nana Banana Pro 或其他AI生图工具生成统一风格的视觉素材：

  • 所有素材生成提示词必须以统一的风格前缀开头
  • 为每个角色生成多张不同角度和状态的参考图
  • 角色辨识度通过不同的颜色方案和视觉标记来区分

风格前缀示例

Chinese ink wash painting style mixed with anime cel-shading

4.4 分镜脚本

生成 Seedance 2.0 时间轴格式提示词，每个分镜脚本包含三部分：

(1) 素材清单

| 素材槽 | 文件 | 说明 |
|--------|------|------|
| 图片1 | C01 | 角色参考 |
| 图片2 | S01 | 场景参考 |

(2) Seedance Prompt（时间轴格式）

水墨武侠风格，9:16竖屏，灰白墨黑色调

0-3秒画面：高空俯拍，场景建立
3-6秒画面：镜头推进，主体引入
6-9秒画面：情节发展，冲突展现
9-12秒画面：高潮时刻，情感爆发
12-15秒画面：结尾落版，氛围延续

【声音】配乐风格 + 音效 + 对白
【参考】@图片1 角色，@图片2 场景...

(3) 尾帧描述

记录本集最后一帧画面，用于下一集衔接。

4.5 视频生成

从第2集开始，使用视频延长功能保持连续性：

  1. 上传上一集视频作为 @视频1
  2. 在提示词开头添加：`将@视频1延长15s`
  3. 这样可以创建各集间的无缝过渡

视频延长流程

片头(新生成) → E1(新生成) → E2(延长E1) → E3(延长E2) → ...

6. 使用技巧

运镜关键词

效果 关键词
推近 推镜头/缓慢推近/急速推进
拉远 拉镜头/逐渐拉远
左右 左摇/右摇/横移
跟随 跟随镜头/跟拍
环绕 环绕镜头/360度旋转
升降 升镜头/降镜头/俯冲
特效 希区柯克变焦/一镜到底
手持 手持晃动

重要限制

  • 最多9张图片：每次Seedance 2.0生成最多支持9张参考图片
  • 最多3个视频：每次最多3个视频作为参考，总时长15秒
  • 敏感词限制：某些词汇可能导致生成失败，如"江湖人士"等
  • 视频编辑能力有限：大多数修改需要重新生成，不能直接编辑
  • 指令遵循：复杂提示词（300+字）可能不一致

文件命名规范

文件类型 命名格式 说明
剧本 [标题]_剧本.md 完整四幕结构剧本
素材清单 [标题]_素材清单.md 编号的素材生成提示词
分镜脚本 [标题]_E[集数]_分镜.md 单集分镜脚本

7. 常见问题

Q: 如何处理敏感词问题？

A: Seedance平台不会明确告知哪个词触发屏蔽。建议使用二分法排查：将提示词分段测试，逐步缩小范围。积累常见敏感词库，在生成时直接规避。

Q: 如何保证角色一致性？

A: 为每个角色生成多张不同角度和状态的参考图，在分镜脚本中明确指定使用哪张图片作为参考。使用统一的风格前缀。

Q: 视频延长后过渡不自然怎么办？

A: 在提示词中显式添加转场指令，如"画面渐暗过渡，天色从夜晚转为清晨"。

Q: 支持哪些风格？

A: 支持任何可以文字描述的风格，如水墨武侠、港漫风格、欧洲古典写实等。关键是使用统一的风格前缀。

Q: 可以生成多长的视频？

A: Seedance 2.0 单次生成最长15秒。多集视频可以通过延长功能串联，理论上可以无限延长。
