1. 项目介绍

A股智能分析系统是一款专业的A股投资分析平台，旨在通过AI技术革新投资分析工作流程。该项目利用CrewAI、AKShare、Python、LangChain、Ollama、Poetry、Pandas等技术，为用户提供从实时行情到财务分析的全方位投资决策支持。

核心特性

  • 📊 全面A股数据：实时行情、历史K线、财务报表、资金流向、行业板块、市场情绪等多维度数据获取
  • 🤖 多Agent协作：4个专业化AI角色协同工作：市场分析师、财务专家、情绪研究员、投资顾问
  • 🇨🇳 A股特色优化：针对中国股市特点：涨跌停分析、政策影响、散户情绪、资金轮动规律
  • 📈 技术面分析：K线形态、均线系统、技术指标、趋势分析、支撑阻力位等专业技术分析
  • 💰 财务面分析：财务比率、趋势分析、同业对比、盈利能力、偿债能力、成长性深度分析
  • 🎯 智能投资建议：综合多维度分析，提供投资评级、目标价位、策略建议、风险控制方案

2. 安装配置

在开始使用 A股智能分析系统 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 8GB RAM（推荐 16GB 或以上）
  • 存储空间：至少 1GB 可用空间
  • 其他：Python 3.12+, Poetry (推荐) 或 UV 包管理器, Ollama (本地) 或 OpenAI GPT

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/easy_investment_Agent_crewai
cd easy_investment_Agent_crewai
  2. 安装依赖（推荐使用 Poetry）： 
    poetry install --no-root
  3. 配置环境变量： 
    cp .env.example .env
# 编辑 .env 文件配置必要参数
  4. 运行分析（默认分析腾讯控股）： 
    poetry run python src/a_stock_analysis/main.py

# 或使用安装的脚本
poetry run a_stock_analysis

6. 常见问题

Q: 如何开始使用 A股智能分析系统？

A: 首先确保已安装Python 3.12+和Poetry，然后克隆项目并安装依赖，配置好环境变量后即可运行分析。

Q: 支持哪些股票市场？

A: 支持A股（上海、深圳交易所）、港股等市场，使用相应的代码格式（如600519.SH、000001.SZ、00700.HK）。

Q: 如何自定义分析参数？

A: 可以通过修改配置文件或在代码中自定义inputs参数来指定分析目标和参数。
