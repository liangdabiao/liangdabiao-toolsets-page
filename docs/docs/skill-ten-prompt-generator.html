1. 项目介绍

Skill Ten - Prompt Generator 是一款先进的AI工具，旨在帮助用户生成高质量的AI提示词。该项目利用 Claude Code、Agent Skills、Markdown、YAML、Prompt Engineering、AI 等技术，为用户提供10个场景化专家的提示词生成解决方案。

核心特性

  • 🎯 自动场景识别：根据用户输入的关键词，自动匹配最合适的专家 Skill
  • 📊 10大场景覆盖：视频生成、图像生成、AI编程、结构化提示词、数据分析、去AI味、创意写作、深度调研、实时语音、长期运行
  • 📋 结构化框架：每个场景都有经过验证的最佳实践框架和模板
  • 🌐 双语支持：完整支持中英文输入和输出
  • 🔗 Claude Code集成：基于Agent Skills技术，直接与Claude对话即可使用
  • 🔄 智能路由系统：关键词自动识别和场景匹配
  • 🎨 专业提示词生成：7层视频结构、S-E-L-C图像框架、TDD编程流程等专业模板
  • 🧩 模块化架构：10个独立Skill，易于扩展和定制

2. 安装配置

在开始使用 Skill Ten - Prompt Generator 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：任何支持现代浏览器的操作系统
  • 网络连接：稳定的互联网连接

安装步骤

  1. 克隆项目仓库：

    cd skill-ten-prompt-generator
  3. 配置Claude Code Agent Skills： 
    # 按照项目文档配置Agent Skills
# 将Skills文件导入到Claude Code环境中

4. 功能详解

Skill Ten - Prompt Generator 包含以下核心功能模块：

10大场景专家

  • 视频生成专家：专业的视频生成提示词优化
  • 图像生成专家：AI生图提示词优化，支持各种艺术风格
  • AI编程专家：代码生成和技术问题解决的提示词优化
  • 结构化提示词专家：JSON格式的结构化输出提示词设计
  • 数据分析专家：数据分析和可视化的提示词优化
  • 去AI味专家：让AI生成的内容更自然、更人性化
  • 创意写作专家：创意写作和角色扮演场景的提示词
  • 深度调研专家：深度研究和搜索的提示词优化
  • 实时语音专家：实时语音对话的提示词设计
  • 自主智能体专家：长期运行与自主智能体的提示词工程

智能路由系统

  • 关键词自动识别
  • 场景匹配算法
  • 动态专家分配

6. 常见问题

Q: 如何确保系统选择了正确的专家？

A: 在描述需求时尽量使用明确的关键词，如"图像"、"视频"、"编程"等，系统会根据关键词自动匹配最合适的专家。

Q: 可以自定义专家的行为吗？

A: 可以通过修改Skills文件来自定义专家的行为和输出模板，以满足特定需求。

Q: 提示词生成的效果如何评估？

A: 通过在实际AI应用中测试生成的提示词，观察输出质量和相关性，根据结果进一步优化提示词。
