1. 项目介绍

简历匹配器是一个AI驱动的平台，旨在逆向工程招聘算法，向您展示如何精准定制简历。该项目利用Next.js 15+、React 19、FastAPI、Python 3.12+、TypeScript、Tailwind CSS 4.x、Ollama 0.6.7、SQLite 3.x、OpenAI API、LlamaIndex、SQLAlchemy、Pydantic等技术，为用户提供从简历上传到优化建议的完整解决方案。

核心特性

  • 📄 简历分析：支持上传PDF或DOCX格式的简历进行智能分析，提取关键信息并进行结构化处理
  • 📋 职位描述解析：深度解析职位描述，提取关键要求、技能需求和企业文化特征
  • 🤖 AI驱动洞察：基于职位要求提供个性化简历改进建议，帮助求职者精准优化简历内容
  • 🔑 关键词匹配：识别对ATS（申请人跟踪系统）重要的缺失关键词，提高自动筛选通过率
  • 📊 结构化数据提取：将非结构化的简历和职位数据转换为结构化JSON格式，便于分析和处理
  • 🔒 本地AI处理：支持Ollama本地AI模型，确保数据隐私安全，也可选择OpenAI云端处理

2. 安装配置

在开始使用 简历匹配器 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 1GB 可用空间
  • 其他：Python 3.12+, Node.js 18+, SQLite 3.x, Ollama 0.6.7+

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/resume-matcher-agent-cn.git
cd resume-matcher-agent-cn
  2. 后端设置： 
    cd backend
pip install -r requirements.txt
  3. 环境配置： 
    cp .env.example .env
# 编辑 .env 文件配置AI提供商
LLM_PROVIDER="ollama"
LL_MODEL="gemma3:4b"
EMBEDDING_PROVIDER="ollama"
EMBEDDING_MODEL="dengcao/Qwen3-Embedding-0.6B:Q8_0"
  4. 启动后端： 
    uvicorn app.main:app --reload

    后端服务将在 http://localhost:8000 启动

  5. 前端设置： 
    cd frontend
npm install
# 或使用 yarn
yarn install
  6. 启动前端： 
    npm run dev
# 或使用 yarn
yarn dev

    前端应用将在 http://localhost:3000 启动

  3. 获取优化建议：获得详细的简历优化建议，提高ATS筛选通过率
    • 个性化改进建议
    • 关键词补充方案
    • 格式调整指导

高级功能

对于更复杂的使用场景，您可以：

  • 自定义AI模型参数以获得更精准的分析结果

4. 功能详解

简历匹配器 包含以下核心功能模块：

技术架构

  • 前端层：Next.js 15+, React 19, TypeScript, Tailwind CSS 4.x, Radix UI
  • 后端层：FastAPI 0.115.12, Python 3.12+, SQLAlchemy, Pydantic, Uvicorn
  • AI层：OpenAI API, Ollama 0.6.7, LlamaIndex, MarkItDown, 自定义提示模板
  • 数据层：SQLite 3.x, 本地存储, 异步数据库, 数据迁移, 日志系统

数据库结构

  • resumes：存储原始简历内容
    • 文件名和路径
    • 上传时间
    • 原始文本内容
    • 文件类型
  • processed_resumes：存储结构化简历数据
    • 个人信息
    • 工作经历
    • 教育背景
    • 技能列表
  • jobs：存储原始职位描述
    • 职位标题
    • 公司信息
    • 原始描述
    • 发布时间
  • processed_jobs：存储结构化职位数据
    • 技能要求
    • 经验要求
    • 关键词列表
    • 匹配规则

API接口

  • 职位管理：
    • POST /api/v1/job/upload - 上传并处理职位描述
    • GET /api/v1/job/ - 根据职位ID检索职位数据
  • 简历管理：
    • POST /api/v1/resume/upload - 上传并处理简历（PDF/DOCX）
    • POST /api/v1/resume/improve - 获取简历改进建议

5. 使用示例

以下是一些典型的使用场景和示例：

示例 1：上传简历

通过API上传PDF格式的简历

curl -X POST "http://localhost:8000/api/v1/resume/upload" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@resume.pdf"

示例 2：上传职位描述

上传目标职位的描述信息

curl -X POST "http://localhost:8000/api/v1/job/upload" \
  -H "Content-Type: multipart/form-data" \
  -F "job_description=@job_desc.txt"

示例 3：获取简历改进建议

获取针对特定职位的简历优化建议

curl -X POST "http://localhost:8000/api/v1/resume/improve" \
  -H "Content-Type: application/json" \
  -d '{
    "resume_id": "resume123",
    "job_id": "job456"
  }'

6. 常见问题

Q: 如何开始使用 简历匹配器？

A: 首先确保已安装Python 3.12+、Node.js 18+和必要的依赖包，然后克隆项目并配置环境变量，分别启动后端和前端服务即可开始使用。

Q: 支持哪些简历格式？

A: 系统支持PDF和DOCX格式的简历文件，能够智能提取其中的关键信息并进行结构化处理。

Q: 如何确保数据隐私？

A: 系统支持Ollama本地AI模型处理，所有数据都在本地处理，确保数据隐私安全。也可以选择云端AI服务进行处理。
