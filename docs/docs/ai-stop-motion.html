1. 项目介绍

AI 定格动画生成器是一款先进的AI工具，旨在基于单张图片创建令人惊艳的定格动画。该项目利用Next.js、Gemini Nano、Framer Motion、React、TypeScript等技术，为用户提供从静态图片生成流畅动画的解决方案。

核心特性

  • 🎬 单图生成动画：基于单张图片创建流畅的定格动画，让静态照片变成生动有趣的动态效果
  • 🤖 Gemini Nano 驱动：使用 Gemini Nano AI 生成分离的姿势序列，基于每个姿势生成高质量的动画帧
  • ⚡ 实时进度反馈：流式响应技术实时返回生成进度，支持长时任务处理，提供详细的生成状态信息
  • 🎮 动画播放控制：支持帧间导航、播放/暂停、帧率调节（1-30 FPS），让你完全控制动画效果
  • 📥 GIF 导出功能：多种导出方案（gifencoder → ZIP → 单帧回退），支持生成高质量 GIF 动画
  • 🛠️ 帧管理系统：完整的帧管理功能，支持帧的添加、删除、移动和预览，精细控制每个动画细节

2. 安装配置

在开始使用 AI 定格动画生成器 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 500MB 可用空间
  • 其他：Node.js v16 或更高版本，pnpm 包管理器

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/ai-make-face-meme
  2. 进入项目目录： 
    cd ai-make-face-meme
  3. 安装依赖： 
    npm install -g pnpm
pnpm install
  4. 配置环境变量（如需要）： 
    # 重命名 .env.example 为 .env
# 在 .env 文件中添加你的 Gemini API key
GEMINI_API_KEY=your_gemini_api_key_here

4. 功能详解

AI 定格动画生成器 包含以下核心功能模块：

前端页面 (/src/app/page.tsx)

  • 文件上传与预览：支持拖放上传，显示上传图片预览
  • 动画生成控制：调用后端API生成动画帧，提供实时进度反馈
  • 动画播放器：支持帧间导航、播放/暂停、帧率调节(1-30 FPS)
  • GIF导出功能：多种导出方案(gifencoder → ZIP → 单帧回退)
  • 帧管理：支持帧的添加、删除、移动和预览

后端API (/src/app/api/stop-motion/route.ts)

  • 图像分析：接收用户上传图片，调用AI生成12个姿势序列
  • 流式响应：实时返回生成进度，支持长时任务(800秒超时)
  • 逐帧生成：基于姿势描述，调用Gemini Nano生成图片帧
  • 错误处理：单帧错误不中断整体生成，提供详细错误信息
  • 数据返回：通过流式API向前端传输base64图片数据

6. 常见问题

Q: 为什么生成动画需要很长时间？

A: 动画生成涉及AI处理，每帧都需要单独生成。整个过程可能需要几分钟，请耐心等待。流式响应会显示实时进度。

Q: 如何获得更好的动画效果？

A: 选择清晰、正面的人像图片通常能获得更好的效果。避免过于复杂或模糊的背景。

Q: 生成的GIF质量如何？

A: 应用支持高质量GIF导出，但文件大小可能会较大。可以根据需要调整帧率来平衡质量和文件大小。
