1. 项目介绍

Amazon商品评论分析系统是一款AI驱动的客户之声分析工具，旨在通过对Amazon商品评论进行多维度智能分析，帮助商家和研究人员深入了解用户反馈。该项目利用Flask、Python、OpenAI、Easy Scraper、ECharts、字节豆包等技术，为用户提供从评论爬取到分析结果的完整解决方案。

核心特性

  • 📦 多维度评论分析：从人群场景、功能价值、保障价值、体验价值等4大维度进行深度分析，提取用户真实反馈
  • 🤖 AI智能标签提取：使用大模型自动识别评论中的关键信息，生成结构化的标签体系和分析报告
  • 📊 用户画像构建：分析使用场景、购买动机、价格感知等，构建完整的用户画像和行为模式
  • 💰 产品反馈分析：深入分析产品优缺点、设计与外观、用户期望建议，为产品改进提供数据支持
  • 🚀 一键式操作：从Easy Scraper爬取评论到上传分析，简单几步即可完成完整的VOC分析流程
  • 📈 数据可视化：生成可视化分析报告，支持Excel筛选和多维度数据展示，便于深入分析

2. 安装配置

在开始使用 Amazon商品评论分析系统 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 500MB 可用空间
  • 其他：Python 3.10+, pip 包管理器, OpenAI API Key 或其他兼容OpenAI接口的大模型

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/easy-amazon-voc
cd easy-amazon-voc
  2. 安装依赖： 
    pip install -r requirements.txt
  3. 配置大模型API： 
    # 在 function.py 中配置API密钥
import openai
openai.api_key = "your api key"

# 支持兼容OpenAI接口的大模型
# 目前使用字节豆包，也可配置其他模型
model="doubao-1.5-pro-32k-250115"
  4. 启动系统： 
    # 启动Flask应用
python app.py

# 访问 http://localhost:5000 开始使用

4. 功能详解

Amazon商品评论分析系统 包含以下核心功能模块：

四维分析框架

  • 人群场景 (People Scenarios)：
    • 使用场景：儿童玩乐、成人收藏、节日礼物、汽车爱好者等
    • 购买动机：价格优势、设计喜欢、替代乐高、性价比考虑等
    • 用户需求：功能需求、情感需求、社交需求等
  • 功能价值 (Functional Value)：
    • 产品优点：零件齐全、组装容易、设计精美、功能多样等
    • 产品缺点：零件缺失、零件松动、组装困难、指示不明等
    • 用户建议：优化指示、提升质量、改进包装等
    • 设计外观：整体造型、内部细节、颜色搭配等
  • 保障价值 (Assurance Value)：
    • 物流配送：配送速度、包装完整性
    • 售后服务：补发零件、退款处理、客服响应
    • 售前服务：信息咨询、产品介绍、购买指导
    • 品质保证：质量承诺、售后支持
  • 体验价值 (Experience Value)：
    • 推荐意愿：是否愿意推荐、推荐原因分析
    • 品牌印象：质量认知、价格感知、品牌信任度
    • 感官感受：外观美感、零件质感、使用体验
    • 价格感知：价格合理性、性价比评估

技术架构

  • 前端技术：HTML5, CSS3, JavaScript, ECharts图表库
  • 后端技术：Flask, Python 3.10+, OpenAI API, 字节豆包模型
  • 数据处理：CSV文件处理, AI标签提取, 结构化数据分析
  • 数据源：Easy Scraper浏览器插件, Amazon商品评论数据

6. 常见问题

Q: 如何开始使用 Amazon商品评论分析系统？

A: 可以使用Easy Scraper Chrome浏览器插件爬取Amazon商品评论，然后导出为CSV格式文件上传到系统进行分析。

Q: 支持哪些数据格式？

A: 支持CSV格式文件，特别是通过Easy Scraper插件从Amazon爬取的评论数据格式。

Q: 如何提高分析准确性？

A: 可以通过选择更高级的AI模型、优化提示词、调整分析参数等方式来提高分析准确性。
