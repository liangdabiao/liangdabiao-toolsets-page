1. 项目介绍

AI生成英语播客视频是一款自动化视频生成工具，旨在通过AI技术将文章内容转换为高质量的英语播客视频。该项目利用Streamlit、FastAPI、FFmpeg、ImageMagick、Azure TTS、OpenAI TTS、Whisper、Pexels API、Python 3.11等技术，为用户提供从文章输入到视频输出的完整解决方案。

核心特性

  • 🤖 AI智能生成：支持视频播客文案AI自动生成，也可以自定义文案，智能转换文章为对话内容
  • 🎬 多尺寸视频：支持竖屏9:16 (1080x1920) 和横屏16:9 (1920x1080) 两种高清视频尺寸
  • ⚡ 批量生成：支持批量视频生成，可以一次生成多个视频，然后选择最满意的结果
  • 🎵 语音合成：支持多种语音合成，可实时试听效果，支持中英文两种语言
  • 📜 智能字幕：支持字幕生成，可调整字体、位置、颜色、大小，同时支持字幕描边设置
  • 🎼 背景音乐：支持背景音乐，随机或指定音乐文件，可调节背景音乐音量
  • 🎨 高清素材：视频素材来源高清且无版权，也可以使用自己的本地素材
  • 🔧 多模型接入：支持OpenAI、Moonshot、Azure、DeepSeek、通义千问、Gemini等多种模型

2. 安装配置

在开始使用 AI生成英语播客视频 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 2GB 可用空间
  • 其他：Python 3.11, Conda, FFmpeg, ImageMagick

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/AI-generated-English-podcast-videos.git
cd AI-generated-English-podcast-videos
  2. 创建虚拟环境： 
    conda create -n AI-podcast-videos python=3.11
conda activate AI-podcast-videos
pip install -r requirements.txt
  3. 安装ImageMagick： 
    # Windows: 下载静态库版本，修改config.toml中的路径
# MacOS: brew install imagemagick
# Ubuntu: sudo apt-get install imagemagick
  4. 启动服务： 
    # 启动Web界面
# Windows
webui.bat

# MacOS/Linux
sh webui.sh

# 访问 http://0.0.0.0:8501

# 启动API服务
python main.py

# API文档 http://0.0.0.0:8080/docs

6. 常见问题

Q: 如何开始使用 AI生成英语播客视频？

A: 首先确保已安装Python 3.11、Conda、FFmpeg和ImageMagick，然后克隆项目并创建虚拟环境，安装依赖后启动Web界面或API服务即可开始使用。

Q: 支持哪些AI模型？

A: 支持OpenAI GPT、Moonshot、Azure OpenAI、DeepSeek、通义千问、Google Gemini等多种模型，国内用户推荐使用DeepSeek或Moonshot。

Q: 如何提高视频质量？

A: 可以选择更高质量的AI模型，调整视频参数（分辨率、帧率等），使用更好的背景音乐和视频素材，以及优化字幕样式。
