1. 项目介绍

Claude Code Deep Research 是一款先进的AI工具，旨在实现专业级别的深度研究。该项目利用 Claude Code、Graph of Thoughts、MCP Protocol、WebSearch、WebFetch、Playwright、多智能体协作 等技术，为用户提供类似OpenAI和Google的Deep Research功能解决方案。

核心特性

  • 🧠 Graph of Thoughts 框架：基于图结构的推理路径管理，支持平衡、深度优先、广度优先三种研究模式
  • 🔬 7阶段深度研究流程：问题细化、研究规划、多智能体研究、信息收集、综合分析、引用验证、报告生成
  • 🤖 多智能体架构：Web研究Agent（3-5个）、学术/技术Agent（1-2个）、交叉引用Agent（1个）并行协作
  • 📊 A-E质量评级系统：A级同行评审、B级队列研究、C级专家意见、D级预印本、E级轶事推测
  • 🔍 引用验证机制：每个事实声明必须包含作者/组织、发布日期、来源标题、URL/DOI、页码
  • 📋 结构化输出：生成1-2页执行摘要、20-50页完整报告、数据、图表、来源、研究笔记、附录
  • ⚡ 简单易用：仅需一个命令 /deep-research [主题]，全程自动化完成深度研究
  • ⚙️ 灵活配置：支持全球范围、时间范围、来源类型、报告格式等多种研究参数定制

4. 功能详解

Claude Code Deep Research 包含以下核心功能模块：

Graph of Thoughts 框架

  • 基于图结构的推理路径管理
  • 支持多种研究模式
  • 动态调整研究方向

7阶段深度研究流程

  • 问题细化：明确研究主题和目标
  • 研究规划：制定研究计划和策略
  • 多智能体研究：并行执行研究任务
  • 信息收集：收集相关数据和信息
  • 综合分析：整合和分析收集的信息
  • 引用验证：验证信息来源和准确性
  • 报告生成：生成专业研究报告

多智能体架构

  • Web研究Agent：负责网络信息收集
  • 学术/技术Agent：负责学术和技术资料分析
  • 交叉引用Agent：负责信息验证和交叉引用

6. 常见问题

Q: 研究过程需要多长时间？

A: 研究时间取决于主题复杂度和所需深度，一般研究需要30分钟到数小时不等。

Q: 如何确保研究结果的准确性？

A: 系统采用A-E质量评级系统和引用验证机制，确保信息来源可靠和准确性。

Q: 可以自定义研究参数吗？

A: 可以，支持自定义时间范围、来源类型、地理范围等多种研究参数。
