1. 项目介绍

小红书商业创意验证器 是一款先进的AI工具，旨在帮助创业者通过小红书数据验证商业创意。该项目利用 Python、MCP Protocol、OpenAI、TikHub、小红书API、Agent架构、数据抓取 等技术，为用户提供基于小红书数据的市场验证解决方案。

核心特性

  • 📱 小红书数据抓取：自动抓取相关笔记和评论数据，使用用户输入作为搜索关键词
  • 🧠 AI 内容分析：使用 LLM 分析用户痛点和市场需求
  • 📊 自动化报告生成：生成专业的 HTML 格式市场验证报告
  • 🔗 MCP 服务器架构：基于 Model Context Protocol 的模块化设计
  • 🤝 三智能体协作：Scraper Agent（数据抓取）、Analyzer Agent（数据分析）、Reporter Agent（报告生成）
  • ⭐ 互动评分系统：点赞×1、收藏×2、分享×3、评论×3的加权策略
  • 🔥 热点分析：识别热门话题、痛点和需求趋势
  • 💾 Checkpoint 机制：保存每个阶段的执行状态，支持断点续传

2. 安装配置

在开始使用 小红书商业创意验证器 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 200MB 可用空间
  • 其他：Python 3.7+，小红书API访问权限，OpenAI API密钥

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/XHS_Business_Idea_Validator
  2. 进入项目目录： 
    cd XHS_Business_Idea_Validator
  3. 安装依赖： 
    pip install -r requirements.txt
  4. 配置环境变量： 
    # 创建 .env 文件
# 配置小红书API和OpenAI API密钥
XIAOHONGSHU_COOKIE=your_xiaohongshu_cookie
OPENAI_API_KEY=your_openai_api_key

5. 使用示例

以下是一些典型的使用场景和示例：

示例 1：美妆产品市场验证

验证美妆护肤产品的市场潜力

   python main.py --idea "婴儿辅食产品" --keywords "育儿,辅食,婴儿食品,营养" --limit 200

6. 常见问题

Q: 如何获取小红书API访问权限？

A: 需要通过合法途径获取小红书Cookie或API访问权限，遵守小红书的使用条款和数据政策。

Q: 数据抓取速度慢怎么办？

A: 小红书有反爬虫机制，请遵守合理的请求频率，避免被限制访问。可以使用代理IP或降低请求频率。

Q: 如何提高AI分析的准确性？

A: 提供更具体和相关的搜索关键词，增加数据样本量，调整AI模型参数或使用更高级的模型。
