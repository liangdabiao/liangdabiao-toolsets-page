1. 项目介绍

Simple Claude Deep Research Agent 是一款先进的AI工具，旨在简化多智能体深度研究流程。该项目利用 Claude Code Skills、MCP Protocol、Playwright、WebSearch、WebFetch 等技术，为用户提供小白友好的深度研究解决方案。

核心特性

  • 🎯 三种查询模式：直接查询（1个子代理）、广度优先查询（3个子代理）、深度优先查询（4个子代理）
  • 🤝 多智能体协作架构：研究主导代理统筹规划、研究子代理并行执行、引用代理自动添加引用
  • ⚡ 并行处理能力：支持2-6个子代理同时工作，显著提升研究效率（复杂问题5-10分钟完成）
  • 🔧 智能工具集成：Web搜索、Web Fetch、Playwright MCP自动处理JavaScript动态页面
  • 🔍 来源质量评估：对信息来源进行批判性评估、交叉验证关键事实、识别偏见和推测
  • 📝 自动化引用生成：研究完成后自动添加引用，支持多种引用格式
  • 👶 完全小白友好：无需编程，一行命令启动 deep-research <查询主题>
  • ⚙️ 高度可定制：可编辑skills文件自定义研究策略和代理行为

基本使用

4. 功能详解

Simple Claude Deep Research Agent 包含以下核心功能模块：

多智能体协作架构

  • 研究主导代理：负责统筹规划整个研究流程
  • 研究子代理：并行执行具体的研究任务
  • 引用代理：自动为研究成果添加引用信息

三种查询模式

  • 直接查询：使用1个子代理进行快速研究
  • 广度优先查询：使用3个子代理从多个角度展开研究
  • 深度优先查询：使用4个子代理进行深入细致的研究

智能工具集成

  • Web搜索：获取广泛的信息来源
  • Web Fetch：抓取特定网页内容
  • Playwright MCP：处理JavaScript动态页面

6. 常见问题

Q: 如何选择合适的研究模式？

A: 简单主题使用直接查询，复杂主题使用广度优先或深度优先模式。

Q: 研究过程中出现错误怎么办？

A: 检查网络连接和Claude Code账户权限，确保所有必要组件正常运行。

Q: 如何提高研究质量？

A: 提供明确具体的研究主题，适当调整研究参数，利用交叉验证功能。
