1. 项目介绍

Facebook广告分析器是一个专业的广告数据分析系统，基于 Claude Code Agent Skill，能够自动分析 Facebook Ads Manager 导出的 CSV 数据。该项目利用Python、Pandas、Claude Code Skill、数据分析等技术，为用户提供从广告数据导入到优化建议生成的完整解决方案。

核心特性

  • 🎯 智能评分系统：根据广告目标（互动/转化/流量）应用加权算法，自动计算每个广告的综合得分
  • 📈 三大目标支持：支持互动目标（消息成本）、转化目标（CPA）、流量目标（CTR）三种广告目标
  • 🏷️ 自动评级机制：将广告分为优秀（前20%）、中等（60%）、差（后20%）三个等级，快速定位问题
  • 🔍 智能诊断引擎：自动分析广告表现，识别问题并提供针对性的优化建议和改进方向
  • 📊 数据驱动标准：基于实际数据分布制定判断标准，使用分位数分析确保评估准确性
  • 📝 完整报告生成：生成详细分析结果、分析报告和投放策略方案三个文件，全方位支持决策

4. 功能详解

Facebook广告分析器 包含以下核心功能模块：

智能评分系统

  • 互动目标：
    • 公式：综合得分 = 消息成本(40%) + 消息数量(30%) + CTR(15%) + CPC(10%) + 互动率(5%)
    • 核心关注：单次消息成本
    • 消息对话数量
    • 互动效果评估
  • 转化目标：
    • 公式：综合得分 = 转化率(35%) + CPA(30%) + CTR(15%) + CPC(10%) + 互动率(10%)
    • 核心关注：转化率
    • 单次转化成本（CPA）
    • 转化效果诊断
  • 流量目标：
    • 公式：综合得分 = CTR(40%) + CPC(30%) + CPM(15%) + 互动率(10%) + 覆盖人数(5%)
    • 核心关注：点击率（CTR）
    • 单次点击成本（CPC）
    • 流量质量评估

评级系统

  • 优秀 (🟢)：得分 > 47分，Top 20% 广告
    • 操作建议：增加预算 30-50%，拓展受众，进行 A/B 测试
  • 中等 (🟡)：得分 26-47分，Middle 60% 广告
    • 操作建议：A/B 测试，优化定向，测试新素材
  • 差 (🔴)：得分 < 26分，Bottom 20% 广告
    • 操作建议：暂停或重新设计，更换素材，重新定位

核心指标

  • CTR (🖱️)：点击率 = 点击量 / 展示次数 × 100%
  • CPC (💰)：单次点击费用 = 花费 / 点击量
  • CPM (📺)：千次展示费用 = 花费 / 展示次数 × 1000
  • 转化率 (🎯)：转化比例 = 潜在客户 / 点击量 × 100%
  • 消息成本 (💬)：单次消息费用 = 花费 / 消息对话次数
  • 互动率 (👍)：互动比例 = 公共主页互动 / 展示次数 × 100%

6. 常见问题

Q: 如何开始使用 Facebook广告分析器？

A: 首先从 Facebook Ads Manager 导出 CSV 数据文件，然后通过 Claude Code Agent Skill 与 Claude 对话，提供数据文件和广告目标即可开始分析。

Q: 支持哪些广告目标？

A: 系统支持三种主要广告目标：互动目标（消息成本）、转化目标（CPA）、流量目标（CTR），每种目标都有相应的评分算法。

Q: 如何理解分析结果？

A: 分析结果将广告分为三个等级：优秀（得分>47分）、中等（26-47分）、差（<26分），每个等级都有相应的操作建议。
