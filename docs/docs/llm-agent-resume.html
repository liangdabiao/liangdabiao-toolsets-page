1. 项目介绍

智能简历筛选系统是一款基于大型语言模型的自动化招聘解决方案，旨在通过AI技术革新传统的简历筛选流程。该项目利用OpenAI GPT、LangChain、ChromaDB、FastAPI、Pydantic、PyPDF2、SQLite、Docker等技术，为用户提供从简历上传到候选人匹配的完整解决方案。

核心特性

  • 💬 自然语言交互：HR可以用自然语言描述招聘需求，系统智能理解并转化为结构化的筛选条件，无需复杂的查询语法
  • 🔍 智能信息提取：利用LLM从PDF简历和JD中提取结构化信息，包括技能、经验、学历、薪资期望等关键数据
  • 📊 量化匹配算法：结合语义理解和结构化信息，对候选人进行多维度量化评估，提供客观的匹配度分数
  • 🔥 多层次筛选：实施语义初筛 → 硬性条件过滤 → 综合评分排序的漏斗式流程，确保筛选结果的准确性
  • 📈 候选人分析：为HR提供详细的候选人匹配度分析和建议，帮助做出更明智的招聘决策
  • 🔌 RESTful API：提供完整的Web API接口，易于与现有HR系统集成，支持灵活的二次开发

2. 安装配置

在开始使用 智能简历筛选系统 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 1GB 可用空间
  • 其他：Python 3.8+, pip 包管理器, OpenAI API Key

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/LLM-Agent-Resume.git
cd LLM-Agent-Resume
  2. 创建虚拟环境： 
    python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
  3. 安装依赖： 
    pip install -r requirements.txt
  4. 配置环境变量： 
    cp .env.example .env
# 编辑 .env 文件，添加 OpenAI API Key
OPENAI_API_KEY=your_openai_api_key

4. 功能详解

智能简历筛选系统 包含以下核心功能模块：

智能筛选流程

  1. 简历上传：HR上传PDF简历文件到系统，支持批量上传PDF格式简历
  2. 信息提取：LLM智能解析简历内容，提取技能、经验、学历等结构化信息
  3. 需求输入：HR用自然语言描述招聘需求，例如：寻找3年以上经验的Python工程师
  4. 语义初筛：基于语义理解进行初步筛选，匹配简历内容与招聘需求的语义相关性
  5. 条件过滤：根据硬性条件进行精确过滤，工作经验、学历要求、技能匹配等硬性指标
  6. 综合评分：技能匹配度、经验相关性、薪资期望等综合评分
  7. 结果输出：生成排序后的候选人列表，包含详细分析报告和建议

API接口

  • 上传简历：POST /api/v1/resumes - 上传PDF简历文件进行处理和索引
  • 提交筛选查询：POST /api/v1/queries - 用自然语言描述招聘需求
  • 获取筛选结果：GET /api/v1/results/{query_id} - 根据查询ID获取筛选结果和候选人列表
  • 获取简历详情：GET /api/v1/resumes/{resume_id} - 根据简历ID获取详细信息和解析结果

技术栈

  • AI框架：OpenAI GPT, LangChain, ChromaDB, 向量搜索
  • Web框架：FastAPI, Uvicorn, Pydantic, RESTful API
  • 文档处理：PyPDF2, python-docx, PDF解析, 文本提取
  • 数据存储：SQLite, ChromaDB, 文件系统, 缓存机制
  • 测试工具：pytest, 单元测试, 集成测试, API测试
  • 部署工具：Docker, Docker Compose, 环境配置, 日志系统

5. 使用示例

以下是一些典型的使用场景和示例：

示例 1：上传简历

通过API上传PDF简历文件

curl -X POST "http://localhost:8000/api/v1/resumes" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@resume.pdf"

示例 2：提交筛选查询

用自然语言描述招聘需求

curl -X POST "http://localhost:8000/api/v1/queries" \
  -H "Content-Type: application/json" \
  -d '{
    "query_text": "寻找3年以上经验的Python工程师"
  }'

示例 3：获取筛选结果

根据查询ID获取筛选结果

curl -X GET "http://localhost:8000/api/v1/results/q1w2e3r4"
