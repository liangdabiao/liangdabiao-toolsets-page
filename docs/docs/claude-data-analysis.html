1. 项目介绍

Claude 数据分析助手是一款先进的智能数据分析平台，旨在利用 Claude Code 的强大功能革新数据分析工作流程。该项目利用 Claude Code、Python、R、SQL、JavaScript、Pandas、NumPy、Scikit-learn 等技术，为用户提供从数据探索到可视化分析的完整解决方案。

核心特性

  • 🤖 智能AI子代理：6个专业化AI子代理：数据探索、可视化专家、代码生成器、报告编写器、质量保证、假设生成器
  • ⚡ 直观斜杠命令：简洁的斜杠命令界面：/analyze、/visualize、/generate、/report、/quality、/hypothesis
  • 🔄 自动化工作流：数据验证、智能上下文、可重现分析，完整的自动化工作流程，提升分析效率
  • 📈 多种分析类型：探索性分析、统计分析、预测分析、完整分析，满足不同的数据分析需求
  • 🎨 丰富的可视化：趋势图、分布图、相关性图、对比图，全面的数据可视化解决方案
  • 💻 多语言代码生成：支持Python、R、SQL、JavaScript，生成生产级分析代码

2. 安装配置

在开始使用 Claude 数据分析助手 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 4GB RAM（推荐 8GB 或以上）
  • 存储空间：至少 500MB 可用空间
  • 其他：Claude Code 访问权限，Python 3.7+，以及必要的数据分析库

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/claude-data-analysis
  2. 进入项目目录： 
    cd claude-data-analysis
  3. 配置 Claude Code 环境： 
    # 在 .claude/settings.json 中配置 Claude Code 设置
# 确保 Claude Code 已正确安装并配置
  4. 准备数据存储目录： 
    # 将 CSV、JSON 或 Excel 文件放入 data_storage 目录
mkdir -p data_storage

4. 功能详解

Claude 数据分析助手 包含以下核心功能模块：

AI 子代理系统

  • 数据探索专家：专业的统计分析与模式发现
  • 可视化专家：创建美观、有洞察力的图表
  • 代码生成器：生产级分析代码生成
  • 报告编写器：综合分析报告生成
  • 质量保证：数据验证与质量控制
  • 假设生成器：研究假设与洞察生成

斜杠命令系统

  • /analyze：对数据集进行探索性分析，发现基本模式
  • /visualize：生成全面的数据可视化图表
  • /generate：生成数据清洗的Python代码
  • /report：生成完整的Markdown格式分析报告
  • /quality：检查并清理数据质量问题
  • /hypothesis：基于数据生成研究假设

6. 常见问题

Q: 如何开始使用 Claude 数据分析助手？

A: 首先确保 Claude Code 已正确安装和配置，然后将数据文件放入 data_storage 目录，最后使用斜杠命令开始分析。

Q: 支持哪些数据格式？

A: 支持 CSV、JSON、Excel 等常见数据格式，将文件放入 data_storage 目录即可进行分析。

Q: 如何自定义分析流程？

A: 可以通过组合不同的斜杠命令创建自定义分析流程，也可以修改子代理配置以适应特定需求。
