1. 项目介绍

Claude 数据分析终极版 是一款先进的AI工具，旨在提供全自动化数据分析服务。该项目利用 Claude Code、Skills、Sub-agents、Python、R、SQL、Pandas、NumPy、Scikit-learn、Matplotlib 等技术，为用户提供互联网数据分析的终极解决方案。

核心特性

  • ⚡ 两个核心命令：/do-all 常规数据分析（带人工反馈）、/do-more 互联网全自动化分析
  • 📊 12个专业技能：RFM客户细分、LTV预测、留存分析、漏斗分析、增长模型、A/B测试分析等
  • 🤖 6个智能子代理：数据探索、可视化专家、代码生成、报告撰写、质量保证、假设生成
  • 🌐 互联网场景覆盖：拉新、留存、促活、推荐、转化、A/B测试、用户分析
  • 🔄 全自动多技能分析：智能匹配技能、按最优顺序执行、生成综合HTML报告
  • 💬 交互式工作流：3个人工反馈检查点，确保分析质量和方向正确
  • 🔧 多种代码生成：Python、R、SQL、JavaScript，支持数据清洗、统计分析、机器学习
  • 📈 丰富输出格式：HTML、PDF、Markdown、DOCX，包含20+种图表类型

4. 功能详解

Claude 数据分析终极版 包含以下核心功能模块：

12个专业技能

  • RFM客户细分：基于最近购买时间、购买频率、购买金额进行客户分层
  • LTV预测：预测客户生命周期价值
  • 留存分析：分析用户留存率和流失原因
  • 漏斗分析：分析用户转化漏斗
  • 增长模型：分析用户增长模式
  • A/B测试分析：分析实验结果
  • 用户画像：构建用户特征模型
  • 行为分析：分析用户行为模式
  • 渠道分析：分析获客渠道效果
  • 收入分析：分析收入来源和趋势
  • 产品分析：分析产品使用情况
  • 运营分析：分析运营活动效果

6个智能子代理

  • 数据探索代理：自动探索数据结构和特征
  • 可视化专家：生成专业图表和可视化
  • 代码生成：生成分析所需的代码
  • 报告撰写：撰写分析报告
  • 质量保证：验证分析结果的准确性
  • 假设生成：提出数据驱动的假设

6. 常见问题

Q: 数据格式有什么要求？

A: 支持常见的数据格式如CSV、Excel、JSON等，数据应包含适当的列名和数据类型。

Q: 分析过程需要多长时间？

A: 分析时间取决于数据量大小和分析复杂度，简单分析可能几秒钟，复杂分析可能需要几分钟。

Q: 如何确保分析结果的准确性？

A: 系统包含质量保证代理，会验证分析结果的合理性，同时提供人工反馈检查点。
