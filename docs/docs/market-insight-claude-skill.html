1. 项目介绍

市场洞察 Claude 技能 是一款先进的AI工具，旨在帮助企业将海量信息转化为可落地的商业洞察。该项目利用 Claude Code、Agent Skills、WebSearch、WebFetch、多智能体协作、数据分析 等技术，为用户提供三段式分析框架的市场洞察解决方案。

核心特性

  • 📊 三段式洞察框架：Stage 1 事实调研、Stage 2 结构化洞察、Stage 3 行动机会清单
  • 🤝 多智能体协作：调研 Agent（网络调研 + 专家网络）、洞察 Agent（策略分析）、机会 Agent（商业转化）
  • 🔍 深度事实调研：市场规模、增长率、用户画像、竞争格局、技术趋势、监管环境
  • 🧠 结构化洞察：用户核心需求、最大痛点、解决方案差距、未来趋势、胜负手
  • 🎯 行动机会清单：机会优先级排序（P0/P1/P2）、执行策略、ROI估算、风险提示
  • 📋 三种输出格式：执行摘要版（核心洞察+P0机会）、完整报告版（全流程分析）、演示文稿版（团队分享）
  • 👶 小白友好：一行命令 /market-insight [产品/市场]，全程自动化完成深度洞察
  • 🚀 商业化落地：每个洞察都指向具体行动，每个机会都包含执行路径

4. 功能详解

市场洞察 Claude 技能 包含以下核心功能模块：

三段式洞察框架

  • Stage 1 事实调研：收集市场基础数据和信息
  • Stage 2 结构化洞察：分析和提炼关键洞察
  • Stage 3 行动机会清单：生成可执行的机会建议

多智能体协作

  • 调研 Agent：负责网络调研和专家网络信息收集
  • 洞察 Agent：进行策略分析和洞察提炼
  • 机会 Agent：将洞察转化为商业机会

深度事实调研

  • 市场规模和增长率分析
  • 用户画像和需求分析
  • 竞争格局和趋势分析

6. 常见问题

Q: 分析结果的准确性如何保证？

A: 系统通过多源数据验证、交叉比对和专家网络信息来确保分析结果的准确性。

Q: 可以自定义分析维度吗？

A: 可以在请求中指定特定的分析重点，系统会相应调整分析维度和深度。

Q: 输出的行动机会如何验证可行性？

A: 系统会提供ROI估算、风险提示和执行路径，但建议结合实际情况进一步验证。
