1. 项目介绍

Monica CRM Claude 技能是一款创新的客户关系管理工具，将传统的 CRM 系统与 AI 对话完美结合。该项目基于 Monica 开源 CRM，通过 Claude Code 和 OpenClaw 技术，让用户可以通过自然语言对话轻松管理客户信息、提醒事项和待办任务。

核心特性

  • 💬 对话式CRM管理：通过自然语言对话添加和管理客户信息，无需复杂操作
  • 👤 客户信息记录：轻松记录姓名、电话、地区等基本信息
  • 🔔 智能提醒系统：为客户设置提醒事项，如会议、跟进、生日等重要时间节点
  • 🏷️ 标签管理：为客户添加自定义标签，便于分类和快速识别
  • ✅ 待办事项管理：为客户关联待办任务，确保不遗漏任何重要事项
  • 🎂 生日提醒：记录客户生日并自动设置提醒，贴心服务提升客户体验
  • 🤖 Claude Code集成：完美集成Claude Code和OpenClaw，利用AI记忆功能
  • 🔓 开源免费：基于Monica开源CRM，可自主搭建和定制

灵感来源

受 OpenClaw 作者 Peter 分享的使用场景启发："我在外面的时候，随手发一条语音备忘问它任何事——无论事情很简单还是很复杂，它都能帮我处理。"

Monica CRM Claude 技能正是基于这一理念打造：你在外面随时给你的 clawbot 发零碎信息，AI 自动24小时为你记录和整理，达到一个良好的 CRM 体验。

3. 使用方法

Monica CRM Claude 技能提供了简洁直观的使用方式：

基本使用

# 添加新客户
"新客户，请加入crm，张三，13800138000，北京人"

# 添加提醒
"请给张三的提醒：明天过来吃饭"

# 添加标签
"请再给张三增加标签'VIP客户'"

"继续为张三添加一个todo：后天带他去参观公司"

支持更复杂的客户管理操作：

  • 批量添加客户信息
  • 设置生日提醒和自动化通知
  • 按标签筛选和管理客户
  • 生成客户关系图谱

4. 功能详解

Monica CRM Claude 技能包含以下核心功能模块：

客户管理

  • 添加、编辑、删除客户信息
  • 记录客户基本资料（姓名、电话、地址等）
  • 管理客户标签和分类
  • 查看客户历史记录

提醒系统

  • 设置一次性或周期性提醒
  • 生日自动提醒
  • 跟进任务提醒
  • 会议和约会提醒

待办管理

  • 为客户关联待办任务
  • 设置任务优先级和截止日期
  • 跟踪任务完成状态

6. 常见问题

Q: 这个 CRM 技能支持哪些平台？

A: 目前支持 Claude Code 和 OpenClaw 两个平台，可以利用它们的记忆功能实现持久化客户管理。

Q: Monica CRM 需要自己搭建吗？

A: 是的，Monica 是开源项目，您可以自己部署。也可以使用 Monica 官方提供的托管服务。

Q: 数据存储在哪里？

A: 客户数据存储在您自己部署的 Monica CRM 系统中，完全掌控自己的数据。

Q: 支持批量导入客户吗？

A: Monica CRM 本身支持 CSV 导入，通过对话也可以逐个或批量添加客户信息。

Q: 可以设置不同级别的提醒吗？

A: 可以，支持设置普通、重要、紧急等不同优先级的提醒。
