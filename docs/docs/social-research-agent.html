1. 项目介绍

社媒调研智能体 是一款先进的AI工具，旨在实现社交媒体舆情深度调研。该项目利用 Claude Code、Agent Skills、TikHub API、WebSearch、WebFetch、数据分析、舆情监控 等技术，为用户提供12+主流社交平台的自动化舆情分析和市场调研解决方案。

核心特性

  • 🔗 双技能合体：社媒调研+深度研究两个Claude Code Skills完美结合，实现全自动化调研
  • 🌐 12+平台覆盖：支持抖音、TikTok、小红书、微博、B站、知乎、Instagram、YouTube等主流社交平台
  • 🤖 智能任务编排：AI自动启动Web Search和TikHub API进行多维度深度调研
  • 📊 舆情分析报告：生成包含情感分析、话题趋势、用户画像、竞品对比的专业报告
  • 📈 数据可视化：生成详细的数据分析图表，直观展示调研结果
  • 🤝 多智能体协作：社媒Agent抓取数据、研究Agent深度分析、报告Agent生成结论
  • ⚡ 一键式操作：只需输入调研主题，系统自动完成从数据收集到报告生成的全流程
  • ⚙️ 可定制化：支持自定义调研参数、时间范围、数据源和报告格式

4. 功能详解

社媒调研智能体 包含以下核心功能模块：

双技能合体系统

  • 社媒调研技能：负责社交媒体数据收集
  • 深度研究技能：负责数据分析和报告生成
  • 自动化流程：两技能协同工作，实现端到端调研

12+平台覆盖

  • 中文平台：抖音、小红书、微博、B站、知乎
  • 国际平台：Instagram、YouTube、Twitter、Facebook
  • 其他平台：TikTok、LinkedIn等

智能任务编排

  • 自动启动Web Search获取背景信息
  • 调用TikHub API获取社交平台数据
  • 多维度数据整合分析

5. 使用示例

以下是一些典型的使用场景和示例：

示例 1：品牌舆情监控

2. 系统在相关社交平台搜索讨论和反馈
3. 分析用户画像和需求特征
4. 生成市场机会和风险分析报告

6. 常见问题

Q: 如何提高数据收集的准确性？

A: 使用更具体和相关的关键词，结合平台特定的话题标签，可以提高数据收集的准确性。

Q: 调研报告的时效性如何？

A: 系统可以指定时间范围进行调研，默认情况下会使用最新的数据，确保报告的时效性。

Q: 是否可以比较多个品牌或话题？

A: 可以，系统支持同时调研多个品牌或话题，并生成对比分析报告。
