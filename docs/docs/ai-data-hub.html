1. 项目介绍

AI数据枢纽是一款AI项目适用的数据中心框架，旨在为AI项目开发提供统一的数据基础设施。该项目利用FastAPI、LangChain、LangGraph、MongoDB、PostgreSQL、Chroma、Milvus、Aliyun OSS、Docker等技术，为用户提供从数据摄取到智能检索的完整数据生命周期管理解决方案。

核心特性

  • 📥 Ingestion（数据摄取）：集成多个AI处理API（OpenAI、Tavily Search）和数据抓取工具，支持小红书、微信公众号等多平台数据采集
  • 🔄 Transform（数据加工）：非结构化到结构化的数据清洗和转换，确保数据准确性和一致性，支持向量化和RAG处理
  • 🔍 Explore（探索分析）：多种数据库支持，灵活的数据存储和查询，包括MongoDB文档存储、PostgreSQL关系型数据库等
  • 📊 Retrieve（查询检索）：NL2SQL自然语言查询、LangGraph RAG框架、多模态图片检索，FastAPI RESTful API接口
  • 🖼️ 多模态支持：图片处理和上传到向量数据库，支持文搜图、图搜图功能，阿里云OSS对象存储集成
  • ⚡ 快速部署：Docker容器化部署，Python Conda环境支持，FastAPI自动服务生成，定时任务调度

2. 安装配置

在开始使用 AI数据枢纽 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 8GB RAM（推荐 16GB 或以上）
  • 存储空间：至少 2GB 可用空间
  • 其他：Python 3.10+, Conda, Docker, Docker Compose, OpenAI API Key

安装步骤

  1. 克隆项目仓库： 
    git clone https://github.com/liangdabiao/AI_data_hub
cd AI_data_hub/backend
  2. 环境配置： 
    cp data.env-dev .env
# 编辑 .env 文件，填入必要的API密钥
OPENAI_API_KEY=your_openai_api_key
TAVILY_API_KEY=your_tavily_key
ALIYUN_OSS_ACCESS_KEY_ID=your_oss_key
ALIYUN_OSS_BUCKET_NAME=your_bucket_name
XHS_AUTH_TOKEN=your_xhs_token
GZH_AUTH_TOKEN=your_gzh_token
SCRAPING_INTERVAL=3600
  3. 创建虚拟环境： 
    # Python conda环境
conda create -n env_310 python=3.10
conda activate env_310
  4. 安装依赖： 
    pip install -r requirements.txt
  5. 启动数据库服务： 
    # PostgreSQL
docker run --name my-postgres \
  -e POSTGRES_USER=myuser \
  -e POSTGRES_PASSWORD=mypassword \
  -e POSTGRES_DB=mydatabase \
  -p 5432:5432 \
  -d postgres

# MongoDB
docker run -d --name mongodb -p 27017:27017 mongo

# ChromaDB
docker run -d --name chroma \
  -p 8000:8000 \
  -v chroma-data:/data \
  chromadb/chroma

4. 功能详解

AI数据枢纽 包含以下核心功能模块：

四层数据架构

  • Ingestion（数据摄取）：集成外部API和数据抓取，OpenAI、Tavily Search、小红书、微信等多渠道数据收集
  • Transform（数据加工）：非结构化到结构化的数据清洗和转换，确保数据准确性和一致性，支持向量化和RAG处理
  • Explore（探索分析）：多种数据库支持，灵活的数据存储和查询，数据挖掘和洞察发现
  • Retrieve（查询检索）：NL2SQL自然语言查询、LangGraph RAG框架、多模态检索，FastAPI RESTful API接口

多数据库支持

  • MongoDB：灵活的文档数据库，支持JSON结构嵌入和抓取内容存储
  • PostgreSQL：结构化关系型数据库，高效处理股票市场数据等结构化信息
  • Chroma：向量数据库，基于目录存储嵌入向量和相关元数据
  • Milvus：高性能向量数据库，支持大规模向量检索和相似性搜索
  • Aliyun OSS：阿里云对象存储，报告、处理数据和备份的云端存储
  • Supabase：开源Firebase替代方案，提供数据库和实时功能

AI工具和平台集成

  • OpenAI：GPT-4.1-mini模型集成，支持多种OpenAI兼容接口
  • Tavily Search：强大的搜索API，提供实时网络搜索能力
  • LangChain：LLM应用开发框架，简化AI应用构建
  • LangGraph：StateGraph框架，实现复杂多步骤工作流
  • 小红书：通过tikhub.io集成，采集丰富的社交媒体数据
  • 微信公众号：公众号平台数据抓取，内容分析和处理

6. 常见问题

Q: 如何开始使用 AI数据枢纽？

A: 首先确保已安装Python 3.10+、Conda、Docker等必要组件，然后克隆项目并配置环境变量，安装必要的数据库服务，最后运行服务即可开始使用。

Q: 支持哪些数据库？

A: 支持MongoDB、PostgreSQL、Chroma、Milvus、阿里云OSS、Supabase等多种数据库，满足不同数据存储和查询需求。

Q: 如何扩展数据源？

A: 可以通过修改scraper模块来添加新的数据源，或者通过API集成的方式接入新的数据平台。
