1. 项目介绍

Claude Code股票深度研究代理是一款专业的股票投资尽调系统，基于 Claude Code Deep Research Framework 构建。该项目利用Graph of Thoughts框架、多智能体协作、WebSearch、WebFetch、Playwright等技术，为用户提供从股票研究到投资建议的完整解决方案。

核心特性

  • 🔬 8阶段研究框架：业务基础、行业分析、业务拆解、财务质量、股权治理、市场分歧、估值护城河、综合报告的完整研究流程
  • 🤖 28个并行智能体：7个研究阶段，每阶段4个并行智能体，2-4小时完成完整尽调
  • 🎯 投资风格适配：支持价值投资、成长投资、困境反转、红利投资四种风格，自动调整研究重点
  • 🚦 信号灯评级：基于基本面分析的信号灯评级系统，Strong Buy (🟢🟢🟢) / Hold (🟡🟡🟡) / Avoid (🔴🔴)
  • 🌍 多市场支持：支持A股、港股、美股及其他全球市场，灵活研究框架适配
  • ✅ 质量保证：利润vs现金流交叉验证、同行对比、空头风险识别、引用质量评级系统

2. 安装配置

在开始使用 Claude Code股票深度研究代理 之前，您需要完成以下准备工作：

系统要求

  • 操作系统：Windows, macOS 或 Linux
  • 内存：至少 8GB RAM（推荐 16GB 或以上）
  • 存储空间：至少 2GB 可用空间
  • 其他：Claude Code 访问权限，稳定的网络连接

安装步骤

  1. 确保已开通 Claude Code 服务并具有 Deep Research 功能权限
  2. 准备股票代码或公司名称（支持A股、港股、美股格式）
  3. 配置 Claude Code 环境（推荐使用 Opus 或 Sonnet 模型）

3. 使用方法

Claude Code股票深度研究代理 提供了简单易用的命令行界面，您可以根据需求进行股票研究：

基本使用

在 Claude Code 中执行股票研究命令：

# 启动 Claude Code
claude

# 设置模型（可选但推荐）
/model opus

# 执行股票研究
/stock-research [股票代码或公司名称]

# 示例：
/stock-research 600519              # 贵州茅台 (A股)
/stock-research AAPL                # 苹果公司 (美股)
/stock-research 腾讯 00700.HK        # 腾讯控股 (港股)

高级功能

对于更复杂的使用场景，您可以：

  • 自定义投资风格和研究重点
  • 指定持有周期和风险偏好
  • 调整研究深度和分析维度
  • 比较多家公司或行业分析

4. 功能详解

Claude Code股票深度研究代理 包含以下核心功能模块：

8阶段研究框架

  • 阶段1 - 业务基础：公司事实底座，包括产品、收入构成、客户、价值链、战略定位
  • 阶段2 - 行业分析：行业周期、供需关系、竞争格局、政策影响
  • 阶段3 - 业务拆解：业务分部、盈利引擎、定价能力、经济特征
  • 阶段4 - 财务质量：指标趋势、现金流vs利润、风险信号、同行对比
  • 阶段5 - 股权治理：股权结构、管理层、资本配置、ROIC分析
  • 阶段6 - 市场分歧：多头观点、空头观点、关键争议、验证节点
  • 阶段7 - 估值护城河：护城河评级(0-5)、相对估值、绝对估值、风险分析
  • 阶段8 - 综合报告：信号评级、投资论点、监控清单

投资风格适配

  • 价值投资：
    • 关注内在价值与安全边际
    • 强调资产负债表分析
    • 常态化盈利分析
    • 指标：P/B, FCF Yield, 正常化EPS
  • 成长投资：
    • 关注市场规模与竞争定位
    • 强调TAM分析
    • 竞争定位评估
    • 指标：收入增长, 护城河, TAM渗透率
  • 困境反转：
    • 关注流动性、偿债能力与催化剂
    • 强调流动性分析
    • 偿债能力评估
    • 指标：负债比率, 现金跑道, 催化剂时间线
  • 红利投资：
    • 关注股息可持续性
    • 强调股息支付能力
    • FCF生成能力分析
    • 指标：股息率, 派息比率, FCF覆盖倍数

信号灯评级系统

  • 🟢🟢🟢 Strong Buy (强烈买入)：
    • 显著的安全边际
    • 强劲的护城河
    • 有吸引力的估值
  • 🟡🟡🟡 Hold (持有)：
    • 估值合理
    • 安全边际有限
    • 可继续持有
  • 🔴🔴 Avoid (避免)：
    • 估值过高
    • 基本面恶化
    • 风险过大

6. 常见问题

Q: 如何开始使用 Claude Code股票深度研究代理？

A: 首先确保已开通 Claude Code 服务并具有 Deep Research 功能权限，然后在 Claude Code 中使用 /stock-research 命令加上股票代码或公司名称即可开始研究。

Q: 支持哪些股票市场？

A: 支持A股（如600519.SH、000001.SZ）、港股（如00700.HK）、美股（如AAPL、TSLA）等全球主要股票市场。

Q: 研究需要多长时间？

A: 完整的8阶段研究通常需要2-4小时，因为系统会部署28个并行智能体进行深度分析，确保研究的全面性和准确性。
