核心算法原理

1. 初始颜色映射
使用局部max pooling，找到像素频率最高的RGB值，用欧氏距离查找最近的颜色
2. 区域颜色合并
BFS查找欧氏距离小于阈值的邻近单元格，形成区域，统一设置为出现次数最多的色号
3. 背景移除
从边界开始洪水填充，标记所有连通的背景色单元格为"外部"
4. 颜色排除与重映射
排除颜色后，在已存在且未被排除的颜色中智能重映射到邻近色

安装与配置

简单几步，开始拼豆创作之旅

1. 克隆项目

获取项目代码：

git clone https://github.com/liangdabiao/perler-beads-ai.git cd perler-beads-ai
2. 安装依赖

使用npm、yarn或pnpm安装：

npm install # 或 yarn install # 或 pnpm install

常见问题

关于AI拼豆底稿生成器的常见疑问

Q: AI优化功能需要付费吗？
A: AI优化基于火山引擎即梦AI API，需要申请API密钥。具体费用请参考火山引擎官方定价。
Q: 支持哪些图片格式？
A: 支持常见的JPG、PNG等图片格式，可直接拖放或点击选择上传。
Q: 如何自定义调色板？
A: 网站提供自定义调色板编辑器，可以创建和编辑自己的调色板，支持添加、删除、修改颜色。
Q: 导出的图纸可以用于商业用途吗？
A: 可以，生成的图纸完全属于你，可以自由使用和分享。
Q: 如何处理杂色问题？
A: 系统提供颜色合并功能，可以自动合并相似颜色。同时支持手动排除特定颜色，系统会智能重映射到邻近色。
