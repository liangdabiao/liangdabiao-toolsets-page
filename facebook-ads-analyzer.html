智能广告数据分析与优化平台

Facebook Ads Analyzer 是一个专业的广告数据分析系统，基于 Claude Code Agent Skill，能够自动分析 Facebook Ads Manager 导出的 CSV 数据。根据不同广告目标（互动/转化/流量）智能评估广告表现，识别最佳和最差广告，生成详细的诊断报告和优化建议，提供数据驱动的投放策略方案。

智能评分系统

根据不同广告目标动态调整指标权重

💬

互动目标

适用于 WhatsApp 消息、Messenger 对话等

综合得分 = 消息成本(40%) + 消息数量(30%) + CTR(15%) + CPC(10%) + 互动率(5%)
  • 核心关注：单次消息成本
  • 消息对话数量
  • 互动效果评估
🎯

转化目标

适用于表单提交、购买转化等

综合得分 = 转化率(35%) + CPA(30%) + CTR(15%) + CPC(10%) + 互动率(10%)
  • 核心关注：转化率
  • 单次转化成本（CPA）
  • 转化效果诊断
🌐

流量目标

适用于网站访问、应用下载等

综合得分 = CTR(40%) + CPC(30%) + CPM(15%) + 互动率(10%) + 覆盖人数(5%)
  • 核心关注：点击率（CTR）
  • 单次点击成本（CPC）
  • 流量质量评估
🟢 优秀

得分 > 47分

Top 20% 广告

操作建议：增加预算 30-50%，拓展受众，进行 A/B 测试
🟡 中等

得分 26-47分

Middle 60% 广告

操作建议：A/B 测试，优化定向，测试新素材
🔴 差

得分 < 26分

Bottom 20% 广告

操作建议：暂停或重新设计，更换素材，重新定位

核心指标

全面的广告指标计算和评估

🖱️

CTR

点击率

点击量 / 展示次数 × 100%
💰

CPC

单次点击费用

花费 / 点击量
📺

CPM

千次展示费用

花费 / 展示次数 × 1000
🎯

转化率

转化比例

潜在客户 / 点击量 × 100%
💬

消息成本

单次消息费用

花费 / 消息对话次数
👍

互动率

互动比例

公共主页互动 / 展示次数 × 100%

最佳实践

获取最佳分析效果的实用建议

1

定期分析

建议每周进行一次分析，跟踪广告表现趋势，及时调整策略

2

数据质量

确保 CSV 文件来自同一时间周期，包含完整的广告数据

3

目标明确

始终明确你的广告目标，选择对应的分析模式（互动/转化/流量）

4

行动导向

将分析结果转化为具体行动，持续优化广告表现
