API接口

完整的RESTful API，易于集成和扩展

POST
/api/v1/resumes

上传简历

上传PDF简历文件进行处理和索引

curl -X POST "http://localhost:8000/api/v1/resumes" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@resume.pdf"
POST
/api/v1/queries

提交筛选查询

用自然语言描述招聘需求

curl -X POST "http://localhost:8000/api/v1/queries" \
  -H "Content-Type: application/json" \
  -d '{
    "query_text": "寻找3年以上经验的Python工程师"
  }'
GET
/api/v1/results/{query_id}

获取筛选结果

根据查询ID获取筛选结果和候选人列表

curl -X GET "http://localhost:8000/api/v1/results/q1w2e3r4"
GET
/api/v1/resumes/{resume_id}

获取简历详情

根据简历ID获取详细信息和解析结果

curl -X GET "http://localhost:8000/api/v1/resumes/a1b2c3d4"

安装部署

简单快速部署，支持多种部署方式

💻

本地部署

1

克隆项目

git clone https://github.com/liangdabiao/LLM-Agent-Resume.git
cd LLM-Agent-Resume
2

创建虚拟环境

python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
3

安装依赖

pip install -r requirements.txt
4

配置环境变量

cp .env.example .env
# 编辑 .env 文件，添加 OpenAI API Key
OPENAI_API_KEY=your_openai_api_key
5

启动应用

# 开发模式
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 生产模式
uvicorn app.main:app --host 0.0.0.0 --port 8000
🐳

Docker部署

1

构建镜像

docker build -t resume-screening .
2

运行容器

docker run -d \
  --name resume-screening \
  -p 8000:8000 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/cache:/app/cache \
  -v $(pwd)/chroma_db:/app/chroma_db \
  --env-file .env \
  resume-screening

项目结构

resume_screening/
├── app/                  # 主应用代码
│   ├── api/              # API路由和接口定义
│   ├── core/             # 核心业务逻辑
│   ├── models/           # 数据模型
│   ├── utils/            # 工具函数
│   └── main.py           # 应用入口点
├── data/                 # 数据文件
├── cache/                # 缓存文件
├── chroma_db/            # 向量数据库
├── config/               # 配置文件
├── tests/                # 单元测试
├── requirements.txt      # Python依赖
├── Dockerfile           # Docker配置
└── .env.example         # 环境变量示例
