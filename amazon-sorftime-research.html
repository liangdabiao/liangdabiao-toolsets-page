安装与配置

简单几步，开始智能选品之旅

1. 获取 Sorftime API Key

访问 Sorftime MCP 获取 API Key：https://sorftime.com/zh-cn/mcp

2. 配置 MCP 服务器

更新 .mcp.json 配置文件：

{ "mcpServers": { "sorftime": { "type": "streamableHttp", "url": "https://mcp.sorftime.com?key=YOUR_API_KEY", "name": "Sorftime MCP" } } }
3. 克隆项目

获取项目代码：

git clone https://github.com/liangdabiao/amazon-sorftime-research-MCP-skill.git cd amazon-sorftime-research-MCP-skill
4. 安装 Claude Code Skills

将技能文件导入到 Claude Code 环境：
.claude/skills/amazon-analyse/
.claude/skills/category-selectionion/

5. 开始使用

在 Claude Code 中调用技能：

/amazon-analyse B07PWTJ4H1 US /category-selection "Wireless Earbuds" US

常见问题

关于亚马逊选品全维度分析工具的常见疑问

Q: ASIN 查询不到产品怎么办？
A: Sorftime 数据库覆盖有限，先使用 product_detail 验证 ASIN。如果返回"未查询到对应产品"， 尝试：1) 使用 product_search 搜索相关关键词；2) 确认是正确的亚马逊站点； 3) 确认 ASIN 格式正确（10位字母数字）。
Q: 报告中出现乱码怎么办？
A: Sorftime 返回的中文是 Unicode 转义格式（\u4ea7\u54c1），技能会自动解码。 如果仍有问题，检查 Python 环境是否支持 UTF-8 编码。
Q: Excel 报告无法打开？
A: 确保安装了 xlsxwriter 库：pip install xlsxwriter
Q: 如何创建自定义技能？
A: 项目包含 skill-creator 工具，可以初始化新技能模板或打包现有技能为 .skill 文件。 详见项目文档。
Q: 支持哪些 Sorftime MCP 接口？
A: 支持产品相关（详情、趋势、评论、关键词）、类目相关（搜索、报告、树结构、趋势、关键词）、 TikTok/1688 等多种接口。详见项目文档。
