安装部署

简单快速部署，支持多种运行环境

后端部署

1

克隆项目

git clone https://github.com/liangdabiao/resume-matcher-agent-cn.git
cd resume-matcher-agent-cn
2

后端设置

cd backend
pip install -r requirements.txt
3

环境配置

cp .env.example .env
# 编辑 .env 文件配置AI提供商
LLM_PROVIDER="ollama"
LL_MODEL="gemma3:4b"
EMBEDDING_PROVIDER="ollama"
EMBEDDING_MODEL="dengcao/Qwen3-Embedding-0.6B:Q8_0"
4

启动后端

uvicorn app.main:app --reload

后端服务将在 http://localhost:8000 启动

前端部署

1

安装依赖

cd frontend
npm install
# 或使用 yarn
yarn install
2

启动前端

npm run dev
# 或使用 yarn
yarn dev

前端应用将在 http://localhost:3000 启动

数据库结构

清晰的数据模型设计，支持高效的简历和职位管理

核心数据表

resumes

存储原始简历内容

  • 文件名和路径
  • 上传时间
  • 原始文本内容
  • 文件类型

processed_resumes

存储结构化简历数据

  • 个人信息
  • 工作经历
  • 教育背景
  • 技能列表

jobs

存储原始职位描述

  • 职位标题
  • 公司信息
  • 原始描述
  • 发布时间

processed_jobs

存储结构化职位数据

  • 技能要求
  • 经验要求
  • 关键词列表
  • 匹配规则
